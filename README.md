# リハビリ通知アプリ

椎間板ヘルニアのリハビリ種目を毎日決まった時間に通知するPWA（Progressive Web App）です。

## 機能

- 毎日指定した時刻にブラウザ通知を送信
- 通知時刻はアプリ内から自由に変更可能
- 4種目のチェックリストで当日の実施状況を記録
- 完了数・連続実施日数を表示
- ホーム画面に追加してアプリとして使用可能

## リハビリ種目

| 種目 | 目的 |
|------|------|
| もも裏ストレッチ | 膝裏の筋肉を伸ばす・ほぐす |
| コアレベルVA | インナーマッスルを鍛える |
| 四つ這いクロスエクステンション | 体幹を鍛える |
| ラットプルダウン | 僧帽筋強化・背筋の安定化 |

## 使い方

1. 以下のURLをスマホのSafariで開く
   - GitHub Pages：`https://tanayou2212.github.io/rehab-reminder/`
   - Netlify：`https://sunny-dolphin-291207.netlify.app/`
2. 「通知を許可する」ボタンをタップ
3. Safariの共有ボタン →「ホーム画面に追加」でアプリとして使用可能
4. 毎日22:00に通知が届く（時刻はアプリ内で変更可能）

## 技術構成

- HTML / CSS / JavaScript（フレームワークなし）
- PWA（Service Worker・Web Push通知）
- データ保存：localStorage（端末内）

## 注意事項

- iOSはSafariでホーム画面に追加した後、そのアイコンから起動しないと通知が動作しません（iOS 16.4以降）
- 設定はデバイスごとに保存されるため、PCとスマホで共有されません
